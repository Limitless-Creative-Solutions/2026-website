## 🔧 CUSTOMIZATION

### Change Greeting:
Edit `getGreeting()` function in `Admin.tsx`:
```typescript
const getGreeting = () => {
  const hour = new Date().getHours();
  if (hour < 12) return "Good Morning";
  if (hour < 18) return "Good Afternoon";
  return "Good Evening";
};
```

### Change Stats:
Add/remove stats in `quickStats` array:
```typescript
const quickStats = [
  { 
    label: "Your Stat", 
    value: 100, 
    color: "from-blue-500 to-cyan-500", 
    icon: YourIcon,
    change: "+10 today",
    trend: "up"
  },
  // ... more stats
];
```

### Change Quick Actions:
Add/remove actions in `quickActions` array:
```typescript
const quickActions = [
  { 
    icon: YourIcon, 
    title: "Your Action", 
    description: "Description", 
    color: "blue",
    tab: "yourTab" as TabType
  },
  // ... more actions
];
```
